# 🌾 Farm Risk Detection Agent

## 🚀 Overview

AI-based system to detect crop risks using:

* Deep Learning (CNN)
* Machine Learning (Random Forest)
* Sensor Data

---

## 🧠 Features

* Disease detection from leaf images
* Water stress detection
* Nutrient deficiency detection
* Final risk score generation

---

## ⚙️ Tech Stack

* Python
* TensorFlow
* OpenCV
* Scikit-learn

---

## ▶️ How to Run

1. Open notebook in Google Colab
2. Run all cells
3. Provide sensor data + image

---

## 🏆 Output

* Risk Level (Low / Medium / High)
* Disease Type
* Nutrient Status
