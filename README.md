# 🌾 AI Farm Risk Detection System

## 🚀 Overview
Multi-modal AI system for detecting crop risks using:
- Sensor data (temperature, humidity, soil moisture)
- Image-based disease detection (CNN)

## 🌱 Supported Crops
- Rice
- Wheat
- Cotton
- Sugarcane

## 🧠 Features
- Water stress detection
- Pest/disease probability
- Nutrient deficiency detection
- Real-time simulation

## ⚙️ Tech Stack
- Python
- TensorFlow
- Scikit-learn
- OpenCV

## 📊 Output
- Risk classification
- Severity scoring
- JSON output for multi-agent systems

## 🔥 Innovation
Combines rule-based + ML + deep learning for multi-modal intelligence.

## 🚀 How to Run
1. Open in Google Colab
2. Run all cells
3. System auto-detects risks
